## 🚀 Host on GitHub Pages

1. Create a **public** GitHub repo named `funlearn`
2. Upload all 4 files (`index.html`, `style.css`, `app.js`, `README.md`)
3. Go to **Settings → Pages → Source: main / root → Save**
4. Your site is live at `https://YOUR-USERNAME.github.io/funlearn` in ~2 min
